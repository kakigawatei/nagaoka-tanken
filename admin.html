初回だけ必要な設定

  1. Firebaseコンソール → Authentication → 始める → ログイン方法で Google を有効化
  2. Authentication → Settings → 承認済みドメイン に kakigawatei.github.io を追加
  3. このページで「Googleでログイン」→ 表示されたUIDをコピー
  4. Firestore → ルール に、既存ルールを消さず下を追加して公開（あなたのUIDは3で取得したもの）： 
    match /announcements/{doc} {
  allow read: if true;
  allow write: if request.auth.uid == "あなたのUID";
}
