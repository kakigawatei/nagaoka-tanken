# アバター別・衣装（コスプレ）イラスト依頼（Codex/ChatGPT 画像生成用）その6

今は歴史衣装（山本五十六など）が「男の子ベースの絵」1種しかないため、女の子アバターで衣装を着ると男の子の顔になってしまいます。そこで、各アバターが各衣装を着た絵を用意します。

## 大事なポイント：作るのは「idle（正面立ち）」だけ＝9枚
アプリで実際に画面に映るキャラのポーズは idle（正面立ち）だけです（happy/point は発見演出の裏で切り替わるだけで見えません）。なので **idle ポーズのみ** でOK。
対象アバターは3種（男の子2・女の子ショート・女の子ツインテール／既存の「男の子」はすでに衣装あり）× 衣装3種 ＝ **9枚**。

## 作り方のコツ
2枚アップして組み合わせてもらうのが確実です：
- ①アバターの絵（例 `guide_idle_girl1.png`）＝この子の「顔・髪・体型・ポーズ・構図」を保つ
- ②衣装の絵（例 `guide_idle_isoroku.png`）＝この「服装（衣装）」に着替えさせる
指示例：「①の女の子の顔・髪・体型・立ち方はそのままで、服装だけ②の衣装（海軍軍服）に着替えさせて。透過PNG・全身・同じ構図で。」

## 衣装3種（既存の男の子版と同じデザインに揃える）
- **isoroku（山本五十六）**：紺の海軍軍服（詰襟・金ボタン・肩章）＋軍帽
- **tsuginosuke（河井継之助）**：幕末の武士〜和洋折衷（着物・袴＋洋式ジャケット、刀を差す）
- **torasaburo（小林虎三郎）**：藍色の羽織袴・書生風、書物を持つ

## 対象アバター3種
- **boy2**（男の子・別の髪型／くせ毛）
- **girl1**（女の子・ショートボブ）
- **girl2**（女の子・ツインテール）

- guide_idle_girl1_isoroku.png / guide_idle_girl1_tsuginosuke.png / guide_idle_girl1_torasaburo.png

## 仕様・納品
- 透過PNG、縦長・全身立ち絵、既存キャラと同じ構図・タッチ（探検録の絵本風）。
- 完成PNGは Drive「06_イラスト素材（アバター別衣装）」へ。
- こちらで 341×512 に軽量化して反映。着せ替えたとき（衣装装備中）に、そのアバターの衣装姿が表示されるように実装します。
- happy/point は不要（画面に出ないため）。揃えたい場合のみ後日でOK。
