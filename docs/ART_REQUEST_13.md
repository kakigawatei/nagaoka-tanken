# ART_REQUEST_13：山古志・蓬平の新スポット お宝カード（5枚）

山古志を5スポットに拡張＋蓬平（新地区）を追加したのに伴い、以下5スポットの
**お宝カード**が必要です。現在は仮画像（treasure.png）で公開中なので、
本イラストが届き次第、下記ファイル名で `assets/` に差し替えます。

## 共通仕様（既存のお宝カードと統一）
- タッチ：レトロなポストカード風／水彩・手描きトーン、探検録の世界観。
  既存の `card_aore.png` `card_yamakoshi.png` などと同じ雰囲気で。
- 構図：縦長（3:4）。対象スポットを主役に、周囲に季節や名物を少し添える。
- 文字・ロゴは入れない（アプリ側で名前や解説を表示するため）。
- 枠：既存カードと同じ薄い枠付きでOK。
- 解像度：1024×1536 程度（納品後に 768×1152 へ軽量化して使用）。
- 背景は不透過でOK（既存カードと同じ）。

## 5枚（ファイル名 → 描く対象）

1. **card_toushijou.png … 山古志闘牛場（牛の角突き）**
   闘牛場のアリーナで、2頭の巨牛が角を突き合わせる勇壮な場面。
   勢子（せこ）が牛に寄り添う。山あいの闘牛場、観客の気配。国重要無形民俗文化財。

2. **card_nakayama.png … 中山隧道**
   手掘りの素朴なトンネルの坑口（アーチ）。岩肌にツルハシの跡。
   雪国の山あい。日本一長い手掘りトンネルの重厚さ・歴史を感じる雰囲気。

3. **card_kogomo.png … 木籠メモリアルパーク**
   中越地震で水没した集落の記憶。水辺に残る家屋（屋根まで水に浸かった様子を静かに）。
   悲しくなり過ぎず、復興と語り継ぎの穏やかなトーンで。芋川・山の谷。

4. **card_tanesuhara.png … 種苧原（たねすはら）**
   山あいに幾重にも連なる棚田・棚池の美しい原風景。錦鯉が泳ぐ棚池、里山の集落。
   （余裕があればアルパカを小さく添えても可）。雪解けの季節感。

5. **card_koryu.png … 高龍神社（蓬平）**
   断崖の中腹に鎮座する朱塗りの社殿へ続く長い石段。商売繁盛のパワースポット。
   のぼり旗、山深い蓬平温泉の谷。荘厳で少し神秘的なトーン。

## 差し替え手順（届いたら私が実施）
1. Drive「13_地図（更新用）」or 指定フォルダから curl でDL
2. 768×1152 に軽量化、原本は `assets/original/` に保管
3. `data/pois.json` と index.html インラインの該当POIの `treasure.image` を
   `treasure.png` → `card_○○.png` に変更
4. sw.js の CACHE_VERSION を上げてプリキャッシュに5枚追加
5. ヘッドレス確認 → 公開
