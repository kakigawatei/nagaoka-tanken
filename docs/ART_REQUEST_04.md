# 案内役キャラの「コスプレ（衣装）」差分イラスト依頼（Codex/ChatGPT 画像生成用）その4

案内役キャラ本人が、長岡ゆかりの偉人3人の衣装を着た差分イラストのお願いです。既存の案内役3ポーズ（`guide_idle.png` / `guide_happy.png` / `guide_point.png`）と**同一構図・同一シルエット**で、衣装だけ着せ替えたイラストを作ってください。

今は仮アート（既存スプライトを色替えしただけのプレースホルダ）が入っているので、急ぎではありません。届き次第、同じファイル名で差し替えます。

## 仕様・コツ
- **サイズ・向き**：既存の案内役スプライトと同じ、縦長・全身立ち絵。透過PNG（背景なし）。
- **構図を合わせる**：3ポーズとも今の絵と同じ体勢・顔の向き・カメラ位置にしてください（差し替えたときにガクッとずれないように）。
  - `idle`：正面立ち、笑顔（帽子・肩掛けバッグはそのシルエットのまま、衣装に合わせて自然にアレンジしてOK）
  - `happy`：両手を上げてジャンプ喜び顔
  - `point`：斜め向きで片手を横に指さすポーズ
- 既存の `guide_idle.png` を1枚アップして「このキャラの体格・顔立ちを保ったまま、衣装だけ以下のように変えて」と伝えると揃いやすいです。
- 探検録の挿絵風、手描き風のやわらかいフラットイラスト、クラフト紙のような質感、朱色・藍・金の差し色、レトロな観光ポスター調（既存キャラと同じトーン）。

## 納品・命名規則
- ファイル名は上記の通り `guide_<ポーズ>_<衣装id>.png`（既存の `guide_idle.png` 等の命名に、衣装idのサフィックスを付ける形）。
- 完成PNGは Drive「04_イラスト素材」へ。
- こちらで `assets/original/` に原寸のまま格納し、`node resize_assets.js` で本番用（高さ512px程度）に軽量化してから `assets/` に反映・公開します。
- 3ポーズ×3衣装＝9枚が理想ですが、揃わない場合は `idle`（代表ポーズ）だけでもOK。その場合はホーム画面・タイトル画面ではidle絵を使い回します。
