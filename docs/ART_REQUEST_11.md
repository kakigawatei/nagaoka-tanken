# ガチャ機「ハンドル回転」用の追加素材 依頼（Codex/ChatGPT 画像生成用）その11

ガチャの演出で「カプセルではなくハンドル（クランク）を回す」ようにしたいです。今のガチャ機の絵はハンドルが本体に一体化していて、ハンドルだけをキレイに回せません。そこで、ハンドルを別レイヤーにするための2枚をお願いします。既存の gacha_machine.png と同じ画角・同じ大きさ・同じタッチで揃えてください（重ねて使うため、位置ずれNG）。

- 今の gacha_machine.png から、**クランクのハンドル（アーム＋木の握り玉）だけを消した**もの。
- 中央の真鍮の丸い軸受け（ハンドルが刺さる根元の丸い金具）は残す。アームと握り玉だけ無い状態。
- それ以外（ドームの空ガラス・木の本体・取り出し口など）は今の絵と全く同じ・同じ位置で。透過PNG。

## ② gacha_handle.png（ハンドル単体）
- クランクのハンドル部分だけ（真鍮のアーム＋木の握り玉）。透過PNG。
- **回転の軸（アームの根元）が画像の中心に来るように**描いてください（中心を軸にぐるぐる回します）。根元から握り玉までがまっすぐ横に伸びた向きで。
- ①の軸受けの位置・大きさにピッタリ重なるサイズ感で。

## 使い方
①の本体に②のハンドルを軸受けの位置で重ねて、②を中心軸で回転させます。これで「ハンドルがぐるぐる回る」演出になります。
