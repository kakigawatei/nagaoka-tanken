# ART_REQUEST_14：与板・栃尾・越路・蓬平の新スポット お宝カード

4地区を拡張（与板+4／栃尾+4／越路+3／蓬平+2）。以下13スポットの**お宝カード**を生成する。
アプリ側は当面 `treasure.png`（仮画像）で公開中。本イラストが届き次第、下記ファイル名で `assets/` に差し替える。

> **Codex（画像生成）へ：** 下の「共通スタイル」を全カード共通の作風とし、各カードを**1枚ずつ**、
> 指定の「描く対象」で生成してください（通常版13枚＋豪華版13枚＝計26枚）。ファイル名は指定どおり。

## 共通スタイル（既存カードと厳密に統一）
- 作風：レトロなポストカード風／水彩・手描きトーン。ゲーム「ながおか探検録」の世界観。
  既存の `card_aore.png` `card_yamakoshi.png` `card_teradomari.png` などと**同じ雰囲気・同じ絵師感**で。
- 構図：**縦長 3:4**。対象スポットを主役に、周囲に季節感や名物を少しだけ添える。
- **文字・ロゴ・ラベルは入れない**（アプリ側で名前・解説を表示するため）。
- 枠：既存カードと同じ薄い枠付きでOK。背景は不透過でOK。
- 解像度：**1024×1536** 程度（納品後に 768×1152 へ軽量化して使用）。

## 通常版（13枚）― ファイル名 → 描く対象

### 与板（直江兼続・お船ゆかりの歴史のまち）
1. **card_kanetsugu_museum.png … 兼続お船ミュージアム** … 前立に「愛」の文字を掲げた兜の甲冑（直江兼続の「愛の甲冑」）を主役に、蔵造り風の資料館を背景に。凛とした歴史ロマンの雰囲気。
2. **card_yoita_castle.png … 与板城跡（城山）** … 城山の山頂の郭跡から、眼下に与板の町並みと信濃川を見下ろす構図。木立、石碑、妻お船が茶をたてた「お船清水」の湧水を小さく添える。
3. **card_moto_yoita.png … 本与板城跡** … 静かな山城の土塁・曲輪跡と山道。直江氏の古城の、苔むした落ち着いた風情。
4. **card_rakuzanen.png … 楽山苑** … 豪商の別邸の池泉回遊式庭園。松・石灯籠・池、上品で優雅な和の庭。

### 栃尾（上杉謙信“旗揚げの地”・雁木のまち・ジャンボ油揚げ）
5. **card_tochio_castle.png … 栃尾城跡** … 山頂の城跡から城下町を見下ろす雄大な眺め。「上杉謙信旗揚げの地」の凛とした空気、若武者の気配。
6. **card_akiba_okunoin.png … 秋葉神社 奥の院** … 精緻な木彫刻（石川雲蝶）で埋め尽くされた荘厳な社殿。法衣姿で穏やかに座す謙信公の銅像を添える。厳かで神秘的。
7. **card_tochio_aburage.png … 栃尾のあぶらげ** … 分厚い巨大な油揚げ（長さ22cm級）を主役に、揚げたての湯気、素朴な店先。食欲をそそる名物グルメの一枚。
8. **card_gangi_tochio.png … 栃尾 雁木のまち並み** … 雪国の雁木（庇の続くアーケード状の通り）が連なる古い商店街。しんしんと降る雪、レトロで温かい町並み。

### 越路（もみじ園・酒どころ）
9. **card_momijien.png … もみじ園（巴ヶ丘山荘）** … 燃えるような紅葉に包まれた日本庭園。樹齢を重ねた大きなもみじ、蔵カフェ、秋の夕日。しっとりと美しい。
10. **card_asahi_shuzo.png … 朝日酒造／酒楽の里あさひ山** … 雪国の造り酒屋の趣。軒先の杉玉、並ぶ酒瓶（久保田・朝日山のイメージ）、試飲の情景。
11. **card_shoraikaku.png … 松籟閣** … 近代和風建築の名邸。松林（松籟）に囲まれた瀟洒で格調高い佇まい。

### 蓬平（高龍神社・奥座敷の温泉）
12. **card_yomogihira_onsen.png … 蓬平温泉** … 山あいの温泉宿の湯けむり、渓谷の紅葉、奥座敷のしっとりした風情。硫黄泉の温かな湯。
13. **card_koryu_okumiya.png … 高龍神社 奥宮** … 山上へ続く118段の急な朱塗りの石段、龍の手水舎、のぼり旗。金運・商売繁盛のパワースポットの神秘的な空気。
