# 加盟店 申請フォーム 項目（確定版・コピペ用）

「ながおか探検録」加盟店募集フォームの質問項目。Tally / Googleフォームにそのまま入れられる形。
運用フロー：**申請（このフォーム・無料）→ 運営が審査 → 承認メールにStripe決済リンク → 入金確認 → こちらでゲーム調のピン/カード生成 → `shops.json`更新・デプロイで掲載**。
（フェーズA＝サーバー不要の手動運用MVP。詳細は `BUSINESS_PLATFORM.md`）

---

## フォーム冒頭の説明文（そのまま貼れます）

> **ながおか探検録｜加盟店 申請フォーム**
> 街歩きゲーム「ながおか探検録」にお店を掲載しませんか？
> ゲームを遊ぶ探検者が、お店を巡って特典を受け取れる仕組みです。
> このフォームは申請（無料）です。内容を確認のうえ、担当より掲載プランと決済のご案内をメールでお送りします。
> ※お店のピンやカードは、いただいた写真をもとに運営がゲームの世界観に合わせて制作します。写真はスマホ撮影で大丈夫です。
> 入力の目安：5〜10分

## セクション1：お店の基本情報

1. **店舗名（正式名称）** ／ 必須・短文
2. **店舗名（ゲーム表示用・略称でもOK）** ／ 任意・短文（未記入なら正式名称を使用）
3. **業種・カテゴリ** ／ 必須・単一選択
   - 飲食店（レストラン・食堂）
   - カフェ・喫茶
   - 居酒屋・バー
   - ベーカリー・スイーツ
   - 小売・物販
   - 美容・サロン
   - 体験・レジャー
   - 宿泊
   - その他
4. **住所（都道府県から）** ／ 必須・短文　※地図上の位置（緯度経度）は運営で設定します
5. **営業時間** ／ 必須・短文（例：11:00〜21:00）
6. **定休日** ／ 必須・短文（例：水曜、年末年始）
7. **店舗の電話番号** ／ 必須・電話

19. **特典を出せる期間** ／ 任意・短文（例：通年／2026年内 など）

## Stripe プラン設計案（たたき台・金額は要調整）

月額サブスク。承認メールにプランごとのStripe Payment Link（月額）を貼って誘導。

| プラン | 月額（税別・仮） | 内容 |
|---|---|---|
| **ライト** | ¥3,000 | 地図にお店ピン掲載＋お店ページ（基本情報・写真・紹介文） |
| **スタンダード** | ¥6,000 | ライト＋クーポン1種（来店特典）＋「今日の得するお店」枠に候補入り |
| **プレミアム** | ¥12,000 | スタンダード＋上位表示＋クーポン複数＋簡易分析（発行/利用数の報告）＋イベント・スタンプラリー参加枠 |

- 初期費用：フェーズA（手動制作）は「ピン/カード制作費」を初回のみ別途いただく形も検討可（例：¥5,000）。数店舗の実証中は無料キャンペーンにしてもよい。
- 支払い：StripeのPayment Link（月額）でカード管理・自動継続・領収書・解約までStripe側が対応 → こちら側サーバー不要。
- 契約前提：**利用規約・特商法表記・プライバシーポリシー**を用意してから募集開始（法人相手は特に必須）。

## 運営側の受付〜掲載チェックリスト（1件ごと）

1. フォーム送信のメール通知を受領
2. 内容審査（特典の適正／掲載可否／写真の質）
3. 承認メール送付（プラン確認＋Stripe Payment Link）
4. 入金確認（Stripeダッシュボード）
5. 住所→緯度経度を確定、写真からゲーム調ピン/カードを制作
6. `shops.json` にレコード追加（id / name / category / lat,lng / offer / coupon.redeemPIN 等）
7. デプロイ → 掲載開始をメール連絡（スタッフ用PINも通知）
