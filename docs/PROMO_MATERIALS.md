# 宣伝チラシ・POP用 Codexプロンプト集（アプリDL誘導）

「ながおか探検録」のダウンロードを促す**チラシ／店頭POP／SNS画像**を、Codex(ChatGPT画像生成)で作るためのコピペ用プロンプト。ゲームの世界観（探検記・クラフト紙・手描き水彩・レトロ、擬洋風建築、探検家キャラ＋コンパス、長岡花火）に統一。

> 💡 **文字とQRの扱い（重要）**：画像生成は日本語の細かい文字やQRコードを正確に描けません。方針は2択：
> - **A案（推奨）**：Codexには「イラスト＋大きなタイトル／キャッチ」までを作らせ、**QRコードと細かい文字は後でCanva等で乗せる**。プロンプトでは「QRは白い四角の空きスペースを用意」「本文テキストは入れない」と指示。
> - **B案**：Codexに文字も入れさせる（崩れたら作り直し前提）。
>
> 💡 **QRの中身**：App Store公開後は「App StoreのアプリページQR」。それまでの暫定は、今のWeb版 `https://kakigawatei.github.io/nagaoka-tanken/` のQRでもOK（公開後に差し替え）。
> 💡 **App Storeバッジ**：Appleの公式バッジ（"Download on the App Store"）は Apple の配布素材を使う決まり。生成画像に描かせず、後で公式バッジ画像を貼る。

## キャッチコピー案（好きなものを使う）
- 長岡を歩いて、まちのお宝を見つけよう。
- あなたの街が、冒険のフィールドになる。
- 歩くほど、長岡がすきになる。
- 史跡めぐりが、宝さがしになる位置情報ゲーム。
- 名所に近づくと、お宝発見。図鑑を集めて長岡を旅しよう。

## ① A4チラシ（縦・配布/掲示用）
```
「ながおか探検録」というスマホの位置情報ゲームを宣伝する、A4縦の印刷用チラシのデザインを作ってください。

世界観・画風：古い探検記／旅の記録帳のような、手描き＋水彩＋クラフト紙の質感。温かみのある配色（セピア・深緑・臙脂・金）。レトロで上品。
主役：探検家の男の子キャラ（つばのある帽子＋コンパスを持つ）。背景に長岡の街（擬洋風のレンガ建築、アオーレ長岡、夜空に長岡花火）。宝箱・お宝カード・金貨が散りばめられたワクワクする雰囲気。

レイアウト：
- 上部：大きなタイトルロゴ「ながおか探検録」。その下に小さめのキャッチコピー「長岡を歩いて、まちのお宝を見つけよう」
- 中央：街を歩く探検家キャラのメインイラスト（花火の夜空を背景に）
- 下部：右下に「QRコードを貼るための白い正方形の空きスペース」を1つ用意。その左に「無料でダウンロード」という短い文字と、App Storeバッジを貼るための横長の空き枠
- 余白をしっかり取り、読みやすく。細かい説明文は入れない。文字は日本語。もし文字が崩れそうなら、タイトルとキャッチだけ大きく入れて他は空欄にしてください。

縦長（A4比率・3:4〜1:1.41）、高解像度、印刷向けのくっきりした線。
```

## ② 卓上POP（加盟店の店頭カウンター用）
```
飲食店や小売店のレジ横カウンターに置く、小さな卓上POP（ハガキ〜A5サイズの三角スタンド）のデザインを作ってください。

用途：この店が「ながおか探検録」の加盟店であることを伝え、アプリのダウンロードと来店特典を案内する。
世界観・画風：①と同じ探検記・クラフト紙・手描き水彩トーン。店頭で目を引くよう少し明るめ。
主役：探検家の男の子キャラが、お店を指さして笑顔で案内しているイラスト。吹き出しで「アプリで特典GET！」。

レイアウト：
- 上部：見出し「このお店は『ながおか探検録』加盟店」
- 中央：キャラ＋吹き出し「アプリで来店特典がもらえる！」
- 下部：QRコードを貼る白い正方形の空きスペース＋「無料アプリ」の短い文字
- 縦長〜正方形。文字は日本語で大きく、細かい本文は入れない。崩れそうなら見出しだけ入れて他は空欄。

高解像度、店頭で映える親しみやすいデザイン。
```

## ③ SNS告知画像（正方形）
```
「ながおか探検録」というスマホ位置情報ゲームの、SNS告知用の正方形（1:1）画像を作ってください。

世界観・画風：探検記・クラフト紙・手描き水彩、レトロで温かい配色。夜空に長岡花火、擬洋風建築の街並み、探検家の男の子キャラ（帽子＋コンパス）。
レイアウト：中央に大きくタイトル「ながおか探検録」、下にキャッチ「長岡を歩いて、まちのお宝を見つけよう」。キャラとお宝カード・金貨を配置。QRや細かい文字は不要。タイトルとキャッチは大きく読みやすく。
正方形・高解像度・SNSで目を引く華やかさ。
```

## 仕上げの流れ（Codexのあと）
1. Codexで①〜③のベース画像を生成（気に入るまで再生成）。
2. Canva等で、**本物のQRコード**（App Store or 暫定Web版URL）と、**App Store公式バッジ**、必要な細かい文字（対応エリア＝長岡市、対応OS等）を上に配置。
3. 印刷（チラシ）／スタンド加工（POP）／SNS投稿。
