# 加盟店マネタイズ方針（現行・意思決定メモ）

> このファイルは「ながおか探検録」加盟店マネタイズの**現在の方針と、次に相談・判断すべきこと**を1枚にまとめた意思決定メモ。次にマネタイズを相談するときは、まずここを読む。
> 最終更新：2026-07-24 ／ 決定者：masa

---

## 1. 現行モデル（2026-07-24 確定）

**「通常3プラン」をアンカー価格として提示し、期間限定キャンペーンで大幅割引して"面を取る"（＝まず加盟店数を増やす）**。

### 通常価格（2027年1月以降の予定価格・アンカー）
| プラン | 月額（税込） | 主な内容 |
|---|---|---|
| ライト | ¥3,000 | マップのお店ピン＋お店ページ＋クーポン1種 |
| スタンダード | ¥6,000 | ライト＋お店オリジナル図鑑カード＋来店3回で豪華版カードに昇格（★1〜3）＋「今日の得するお店」候補 |
| プレミアム | ¥12,000 | スタンダード＋店舗独自オリジナルイベント（※）＋複数クーポン＋特別仕様の豪華カード＋スポンサー表記＋上位表示・レポート＋AI導入・活用サポート（※） |

- **加盟金（初期登録料）**：通常 ¥5,000（税込・初回のみ）
- **混雑ランプ（🟢🟡🔴を店頭から発信）**：全プランで**無料**
- ※オリジナルイベント／AI活用サポートは、ヒアリング→企画→アプリのアップデートで順次提供（即時ではない）

### 開店応援キャンペーン（〜2026年12月31日）
| プラン | キャンペーン価格 |
|---|---|
| ライト | **無料**（通常¥3,000/月） |
| スタンダード | **¥3,000/月**（通常¥6,000・半額） |
| プレミアム | **¥6,000/月**（通常¥12,000・半額） |
| 加盟金 | **無料**（通常¥5,000） |

### 契約条件
- **最低利用期間なし＝いつでも解約OK**（以前あった「3か月縛り」は**廃止**。面取り優先のため）
- 支払い：クレジットカード（Stripe・毎月自動）または銀行振込（前払い）。ライトの無料期間中は課金なし。
- すべて税込表示。適格請求書（インボイス）発行事業者ではない。

## 2. 次に判断すること（★これがこのメモの主目的）

**2026年12月31日が近づいたら、その時点の加盟店の状況を踏まえて決める：**

- **A. キャンペーンを延長する**（まだ面が取れていない／勢いを続けたい場合）
- **B. 通常価格に移行する**（十分に加盟が集まり、収益化フェーズに入る場合）
- **C. 価格・プラン構成そのものを見直す**（実際の反応を見て）

判断材料：加盟店数、どのプランが選ばれているか、解約率、クーポン利用の反応、混雑ランプの稼働状況 など。

---

## 3. 未対応の宿題（実務）

- **Stripe 決済リンクがキャンペーン価格に未対応。** 既存の決済リンク3本は旧価格バンドル（ライト¥3,000/スタンダード¥6,000/プレミアム¥12,000＋加盟金¥5,000）のまま。キャンペーン価格で課金するには：
  - スタンダード¥3,000／プレミアム¥6,000の**新しい決済リンクを作る**（加盟金バンドルなし）、または**銀行振込**で対応。
  - **ライトは無料なので決済リンク不要。**
  - → 実際に加盟店が来るタイミングで対応（それまで急がない）。詳細な手順は `docs/STRIPE_SETUP.md`。
