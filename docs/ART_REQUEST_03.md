# 追加地区ぶん — イラスト依頼（Codex/ChatGPT 画像生成用）その3

新しい地区を増やすための「地区マップ＋お宝カード」。1地区につき **マップ1枚＋カード1枚**。作れる分だけでOK（1地区だけでも増やせます）。完成PNGは Drive「03_イラスト素材」へ、下記ファイル名で。

## 仕様・コツ
- **地区マップ**：1024×1536（縦）、俯瞰イラストマップ、背景込み、ピンを乗せるので**余白多め・文字少なめ**（既存の map_chuo / map_yamakoshi と同じ感じ）。
- **お宝カード**：1024×1536（縦）、レトロなポストカード風。既存の card_aore.png を1枚アップして「この画風に合わせて」と言うと揃います。
- 各プロンプト末尾に共通スタイルを付ける：
  > 探検録の挿絵風、手描き風のやわらかいフラットイラスト、クラフト紙のような質感、朱色・藍・金の差し色、レトロな観光ポスター調
