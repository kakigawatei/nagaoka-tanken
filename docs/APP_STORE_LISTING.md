# App Store 申請メタ情報（ながおか探検録）

App Store Connect に入力する内容一式。コピペで使える。

## 基本
- **アプリ名**：ながおか探検録
- **サブタイトル（30字以内）**：長岡を歩いて まちのお宝発見
- **バンドルID**：com.nagaokatanken.app
- **主要言語**：日本語
- **SKU**（任意の管理用ID）：nagaokatanken-001
- **価格**：無料
- **カテゴリ**：主：ゲーム ／ 副：旅行（お好みで。地域観光アプリとしての発見性を狙うなら主を「旅行」でも可）

## プロモーションテキスト（170字以内・後から変更可）
```
長岡の街を歩いて、史跡や名所に近づくとお宝を発見！コインを集めて図鑑をコンプリート。歩く楽しさが、まちを知るきっかけになる位置情報ゲームです。無料で遊べます。
```

## 説明文（Description）
```
新潟県長岡市を舞台にした、歩いて楽しむ位置情報ゲームです。

■ 歩いて、まちのお宝を発見
アオーレ長岡、山本五十六記念館、河井継之助記念館、摂田屋の醸造のまち、山古志の錦鯉、寺泊の魚の市場通り…。長岡の史跡・名所に実際に近づくと「お宝」を発見。集めたお宝は図鑑になり、その土地の歴史や物語も読めます。

■ 集めて、育てて、また行きたくなる
・発見でコインとXPを獲得、レベルアップ
・ショップで便利アイテム、衣装だんすでキャラの着せ替え
・現地で回せる「ご当地ガチャ」とおみやげコレクション
・同じ場所に通うほど、図鑑カードが豪華版に昇格

■ こんな人におすすめ
・お散歩やまち歩きが好きな方
・長岡の歴史・文化にふれてみたい方
・観光やおでかけのきっかけがほしい方

■ 位置情報について
位置情報は史跡への接近判定にのみ使い、端末の中だけで処理します。外部に送信することはありません。

さあ、探検家になって長岡を巡りましょう。あなたの街が、冒険のフィールドになります。
```

## 年齢区分（レーティング）
- 不適切な要素は無し → **4+**（全年齢）。レーティング質問はすべて「なし／No」でOK。

## App Privacy（データ収集の質問）
本アプリは端末内で完結し、利用者データを外部に送信・収集しません。
- 「データを収集していますか？」→ **収集していない（Data Not Collected）** を選択でOK。
  - 位置情報は端末内で判定に使うだけで、収集・送信・第三者提供をしていないため。
- ※もし審査で位置情報の用途を聞かれたら「史跡への接近判定にのみ使用、端末内処理、外部送信なし」と回答。

## スクリーンショット（必須）
- **6.7インチ（iPhone）用**が最低限必須（例：iPhone 15/16/17 Pro Max のサイズ）。3〜5枚あると良い。
- おすすめの画面：
  1. ホーム（探検家キャラ＋コイン＋「今日はどこへ探検に行こうか！」）
  2. マップ（地区タブ・ピン）
  3. お宝発見の演出 or 図鑑（豪華版カードが映える）
  4. ガチャ（カプセルドーム）
  5. 図鑑グレード／おみやげ or 衣装だんす
- 撮り方：実機でスクショ（電源＋音量ボタン）or シミュレータで撮影。ステータスバーがキレイな状態で。
