# キャラの「見た目（アバター）」バリエーション イラスト依頼（Codex/ChatGPT 画像生成用）その5

主人公キャラの「性別・髪型ちがい」バージョンのお願いです。**服装・持ち物・体型・構図・タッチは今のキャラとまったく同じ**にして、**髪型と顔の性別的な印象だけ**を変えてください。差し替えたときにピタッと合うように、既存の `guide_idle.png` / `guide_happy.png` / `guide_point.png` を参照用にアップして「このキャラの服・体型・構図はそのままで、髪と顔だけ変えて」と伝えるのがコツです。

## 今のキャラ（共通の見た目＝必ず踏襲）
- 茶色のサファリ帽（探検帽）
- クリーム色（うすい黄）の半袖シャツ
- オレンジ（朱色）の三角スカーフを首に
- 茶色の斜めがけショルダーバッグ（ストラップが胸を斜めに横切る）
- 茶色の半ズボン、クリーム色のハイソックス、茶色のブーツ
- まる顔でかわいい、ほっぺにピンクの赤み、太めの茶色い輪郭線、やわらかいフラット塗り、探検録の絵本風トーン
- 透過PNG（背景なし）、縦長・全身立ち絵

## 3ポーズ（今の絵と同じ体勢・顔の向き・カメラ位置）
- **idle**：正面立ち、手は自然に下ろす、やさしい笑顔
- **happy**：両手を上げてジャンプ、目がキラキラ（星型）、大きく口を開けた笑顔
- **point**：斜め向き、片手を腰に当て、もう片方の手で横を指さす、笑顔

## 納品・命名規則
- ファイル名は上記の通り `guide_<ポーズ>_<アバターid>.png`（既存 `guide_idle.png` 等に、アバターidのサフィックスを付ける形）。
- 完成PNGは Drive「05_イラスト素材（アバター）」フォルダへ。
- こちらで `assets/original/` に原寸保管し、既存スプライトと同じ 341×512 に軽量化して `assets/` に反映・公開します。
- 3ポーズ×3アバター＝9枚が理想。揃わない場合は各アバターの `idle` だけでもOK（その場合は全ポーズで idle 絵を使い回します）。
