## 与板（Yoita）― 直江兼続・お船ゆかりの歴史のまち／越後与板打刃物
既存：与板 都野神社
候補：
1. **与板歴史民俗資料館（兼続お船ミュージアム）** … 兼続の「愛の甲冑」（複製）を360度展示、甲冑着用体験。与板出身で“日本人初のビール醸造人”中川清兵衛の展示も。地域のランドマーク。
2. **与板城跡（城山）** … 直江兼続の居城。山頂に実城・二ノ郭・三ノ郭。妻お船が茶をたてた「お船清水」。
3. **本与板城跡** … 直江氏の元の居城。「愛と義」お船生誕の地。
4. **楽山苑（らくざんえん）** … 豪商・大坂屋三輪家が造った庭園。
（＋物産系：越後与板打刃物＝伝統的工芸品／250年続く「登り屋台」祭り）

## 栃尾（Tochio）― 上杉謙信“旗揚げの地”・雁木のまち・ジャンボ油揚げ
既存：栃尾 秋葉神社
候補：
1. **栃尾城跡** … 上杉謙信（長尾景虎）14歳で入城・旗揚げの地。城山。
2. **秋葉公園（謙信公銅像・奥の院）** … 法衣姿の謙信公銅像。奥の院は石川雲蝶＆熊谷源太郎の彫刻で埋め尽くされ市文化財。※既存「秋葉神社」と同エリア＝「奥の院・雲蝶彫刻」として差別化推奨。
3. **栃尾の油揚げ（あぶらげ）** … 長さ22cm級のジャンボ油揚げ。揚げたてグルメスポット（お店連携にも好相性）。
4. **雁木のまち並み** … 雪国の歴史的町並み。

## 越路（Koshiji）― もみじ園・酒どころ
既存：宝徳山稲荷大社
候補：
1. **もみじ園（巴ヶ丘山荘）** … 明治期の庭園、樹齢150〜200年のもみじ、秋は紅葉ライトアップ、蔵カフェ「髙九蔵Cafe」。国登録有形文化財。看板スポット。
2. **朝日酒造／酒楽の里あさひ山** … 「久保田」「朝日山」の酒蔵。全種試飲バー、食事処「あさひ山蛍庵」。
3. **松籟閣（しょうらいかく）** … 朝日酒造創業家の邸宅、文化財級の近代和風建築。

## 蓬平（Yomogihira）― 高龍神社・奥座敷の温泉
既存：高龍神社
候補：
1. **蓬平温泉（よもぎひら温泉）** … 「長岡の奥座敷」。単純硫黄泉で肌つるつる、旅館街（福引屋・和泉屋等）。600年以上前、龍神伝説が起源。
2. **高龍神社 奥宮／龍の手水舎・118段の石段** … 既存の高龍神社の“奥”。商売繁盛・金運のパワースポットの参道演出。
※蓬平は資源が高龍神社＋温泉に集中＝現実的には+1〜2スポット。近隣（太田の棚田等）まで広げる手も。

## 進め方（案）
1. masaさんが各地区で「入れるスポット」を選ぶ（寺泊同様、各3〜5個が目安）。
2. こちらで緯度経度を確定→ `data/pois.json` に追加＋地図bbox（MAP_GEO）調整。
3. 地図：必要なら該当地区を寺泊同様に実地形＋トーン統一で再生成（[[MAP_REALISM]]）。
4. カード絵：Codexで各スポットの通常版／豪華版を制作（既存のART_REQUEST方式）。
