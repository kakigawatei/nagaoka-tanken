## 2. 投稿の型は5つだけ（毎回ゼロから考えない）

**型を決めておけば、ネタ出しに悩まなくなる。** 使い回す素材も決めておく。

### 型A：スポット紹介（王道・週1）
実在のスポット写真 → ゲーム内のお宝カード画像を並べる。「現実とゲームがつながる」のが一番の売り。
- 素材：現地写真（スマホ可）＋ `assets/card_*.png`（**84枚もうある**）
- キャプション例：
```
【#12 機那サフラン酒本舗】
思わず二度見する、極彩色の"鏝絵の蔵"。
明治の実業家が財を注ぎ込んだ、長岡が誇る奇跡の建物です。

ながおか探検録では、ここを訪れると
お宝カード「鏝絵の蔵」が手に入ります🎴

📍摂田屋エリア（長岡駅から車で10分）
🎮 アプリはプロフィールのリンクから（無料）

#長岡 #摂田屋 #新潟観光 #ながおか探検録
```

### 型B：お宝カード見せ（映え・週1）
カード画像だけで成立する。**通常版→豪華版の変化**は"引き"が強い。
- 「**3回通うとカードが豪華版になります**」＝リピートの仕掛けをそのまま見せる（加盟店にも刺さる）
- 素材：`card_xxx.png` → `card_xxx_deluxe.png` の**ビフォーアフター**

### 型C：リール（拡散担当・週1）
**フォロワーを増やすのはリールだけ**と割り切る。写真投稿は既存フォロワー向け、リールは新規向け。
- 尺は**15〜25秒**。最初の1秒で「長岡」と分かる画を出す。
- 構成テンプレ：
  1. 現地の映像（3秒・歩きながら）
  2. テロップ「ここ、ゲームの中では"お宝"です」
  3. お宝カードがバーンと出る演出
  4. 「長岡を歩くだけのゲーム、あります」→ プロフィールへ
- 音源は**流行りの音を必ず使う**（無音・自前BGMは伸びない）

### 型D：加盟店紹介（営業の武器・月2〜4）
加盟店ができたら必ず投稿する。**これがあるから加盟店が集まる**。
- 「加盟店になると、こう紹介されます」というサンプルにもなる
- 既に実在の加盟店：**東京油そば 柿川亭**（デモ枠を実店舗に）

### 型E：中の人（信頼づくり・不定期）
「個人で作ってます」は**強い武器**。開発の裏側、アップデート報告、失敗談。
- 例：「App Store審査、通りました」「40スポット目、追加しました」
- 地方creator/個人開発は応援されやすい。**加盟店の社長さんが見て「この人なら」と思う**のはこの型。

## 8. KPI と「うまくいってるか」の判断基準

**フォロワー数だけ見ない。** 見るべきはこの3つ：

| 指標 | どこで見る | 合格ライン |
|---|---|---|
| **保存数** | 投稿のインサイト | いいねの**5〜10%**あれば良コンテンツ（＝後で行きたい、と思われた） |
| **プロフィールアクセス→リンククリック** | インサイト | **これがアプリDLに直結**。週10クリックあれば上出来 |
| **リールの視聴維持率** | リールのインサイト | 最初の3秒で50%以上残っていればOK |

**判断**
- 1か月やって**リールが全部300再生以下**なら、**内容ではなく「冒頭1秒」を疑う**（音源・テロップを変える）
- **保存が伸びる投稿の型が見つかったら、そればかり作る**（新しいことをやらない勇気）

## 9. 加盟店営業との接続（ここが本命）

1. **加盟店を紹介する投稿を必ず出す**（型D）
2. その投稿を**営業のときに見せる**：「加盟いただくと、こういう形で紹介します」
3. **プレミアムプランの「スポンサー表記」**（[[MERCHANT_APPLICATION_FORM]]）と連動させる
   → イベント告知投稿に「協賛：◯◯様」を入れる＝**Instagramがプレミアムの価値を裏付ける**
4. フォロワーが増えるほど**加盟店の月額が正当化される**（＝値上げ余地も生まれる）

> **提案**：営業資料（`docs/merchant/`）に「**公式Instagramでもご紹介します**」の1行を足すと、成約率が上がる。フォロワー数が3桁になったら追記する。

## 10. やらないこと（疲弊防止・これ大事）

- ❌ **毎日投稿しようとする** → 100%続かない。週3で十分
- ❌ **フォロワーを買う／相互フォロー** → 加盟店に見せる数字の信頼性が死ぬ
- ❌ **他SNSに同時展開**（X・TikTok・YouTube）→ 1つも育たない。**まずInstagram1本**に絞る
  - ※リールは**そのままTikTok/YouTube Shortsに転載できる**ので、余裕が出たら「投稿するだけ」で横展開すればいい
- ❌ **完璧な写真を撮ろうとする** → スマホで十分。**現地に立った臨場感**の方が価値がある
- ❌ **アプリの宣伝ばかりする** → 「宣伝アカウント」と見なされた瞬間フォローされない。**9割は長岡の魅力、1割がアプリ**
